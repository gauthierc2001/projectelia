# Project Elia

Welcome to Elia's World - A beautiful Next.js website showcasing Project Elia.

## Features

- Modern Next.js 15 with TypeScript
- Tailwind CSS for styling
- Responsive design
- Beautiful gradient backgrounds
- Interactive elements

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This project is ready for deployment on Vercel. Simply connect your GitHub repository to Vercel for automatic deployments.

## Technologies Used

- Next.js 15
- TypeScript
- Tailwind CSS
- React

## Project Structure

- `src/app/` - Next.js app directory
- `public/` - Static assets
- `src/app/page.tsx` - Main page component
- `src/app/layout.tsx` - Root layout component
